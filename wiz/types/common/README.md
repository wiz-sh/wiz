# `@types/common`

Import only the command families a project uses: `@types/common/git`,
`@types/common/docker`, `@types/common/search`, `@types/common/text`, or
`@types/common/systemd`.

The `search` entry provides option-aware grep and ripgrep contracts. `text`
contains fd, fzf, bat, less, tree, awk, sed, jq, and archive utilities.
