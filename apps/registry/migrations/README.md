# Registry migrations

Migrations are append-only SQL files generated by Drizzle Kit and applied by the Bun SQL Drizzle
migrator. Applied hashes are recorded in `drizzle.registry_migrations`; changing an applied file
is unsupported. Production deployments should run `bun run migrate` once before replacing API or
worker containers.

Use `bun run db:generate` after changing `src/database/schema.ts`, then review both the generated
SQL and schema snapshot before committing them.
